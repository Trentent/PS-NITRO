formation_topology

Configuration for Formation Topology resource.

Read/write properties

template_name<String>

Formation Template Name.

entity_ip<String>

Entity IP Address.
Minimum length = 1
Maximum length = 64

connections<String>

Connections.

entity_state<String>

Entity state.

version<String>

Template version.
Minimum length = 1
Maximum length = 128

entity_name<String>

Entity Name.
Minimum length = 1
Maximum length = 256

formation_instance_id<String>

Formation Instance ID.
Minimum length = 1
Maximum length = 128

entity_id<String>

Entity ID.
Minimum length = 1
Maximum length = 256

entity_type<String>

Entity Type.
Minimum length = 1
Maximum length = 128

id<String>

Id is system generated key for all the Formation Instances.

Read only properties
