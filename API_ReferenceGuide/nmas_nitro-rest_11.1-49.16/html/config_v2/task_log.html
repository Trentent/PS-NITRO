task_log

Configuration for Task Log resource.

Read/write properties

tenant_id<String>

Tenant Id of the task log.
Minimum length = 1
Maximum length = 128

id<String>

Id is system generated key for all the task logs.

Read only properties

executed_by<String>

User Name.

status<String>

Status.

starttime<Integer>

Start Time.

name<String>

Task Name.

endtime<Integer>

End Time.
