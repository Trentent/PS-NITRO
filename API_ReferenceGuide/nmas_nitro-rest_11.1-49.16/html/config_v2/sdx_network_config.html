sdx_network_config

Configuration for SDX Network Configuration resource.

Read/write properties

dns<String>

DNS Server.
Minimum length = 1
Maximum length = 64

gateway<String>

Gateway .
Minimum length = 1
Maximum length = 64

svm_ipv6_address<String>

Management Service IPv6 Address.

config_type<Integer>

Configuration Type. Values: 0: IPv4, 1: IPv6, 2: Both.
Maximum value =

svm_ip_address<String>

Management Service IP Address.
Minimum length = 1
Maximum length = 64

dns_v6<String>

DNS Server IPv6 Address.

gateway_ipv6<String>

Gateway IPv6 Address.

xen_ip_address<String>

XenServer IP Address.
Minimum length = 1
Maximum length = 64

netmask<String>

Netmask.
Minimum length = 1
Maximum length = 64

network_interface<String>

Interface on which management needs to be enabled.
Minimum length = 1
Maximum length = 15

init_status<Integer>

System initialize status.

Read only properties

act_id<String>

Activity Id.
