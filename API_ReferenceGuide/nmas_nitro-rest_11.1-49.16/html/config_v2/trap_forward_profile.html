trap_forward_profile

Configuration for Trap Forward Profile resource.

Read/write properties

trap_destination_port<String>

SNMP port to forward trap .
Minimum length = 1
Maximum length = 128

trap_community<String>

SNMP community..
Minimum length = 1
Maximum length = 128

id<String>

Id is system generated key for all the mail profile..

trap_destination<String>

SMTP server name.
Minimum length = 1
Maximum length = 128

profile_name<String>

Profile name for the mail setting..
Minimum length = 1
Maximum length = 128

Read only properties

tenant_id<String>

Tenant Id .
