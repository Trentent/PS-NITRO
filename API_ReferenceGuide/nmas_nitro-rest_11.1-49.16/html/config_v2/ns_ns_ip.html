ns_ns_ip

Configuration for NS IP on NetScaler resource.

Read/write properties

save_config<Boolean>

true, if save config is required.

type<String>

Type of IP Address, Possible Values: SNIP.
Minimum length = 1
Maximum length = 64

netmask<String>

netmask.
Minimum length = 1
Maximum length = 64

ipaddress<String>

NS ipaddress that needs to be created.
Minimum length = 1
Maximum length = 64

ns_ip_address<String>

NetScaler IP Address where NSIP needs to be created.
Minimum length = 1
Maximum length = 64

Read only properties

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.

The following parameters can be used in the nitro request :

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
