app_template_info

Configuration for App Template Info resource.

Read/write properties

downloads<String>

Number of downloads for the template.

version<String>

Version Supported.

name<String>

Template Name.

comments<String[]>

Comments for the template.

rating<String>

Rating for the template.

updated_on<String>

Updated date of the template.

desc<String>

Description for the template.

download_link<String>

Download link for the template.

Read only properties
