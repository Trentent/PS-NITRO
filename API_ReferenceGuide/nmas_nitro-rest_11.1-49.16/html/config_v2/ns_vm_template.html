ns_vm_template

Configuration for NS VM Template resource.

Read/write properties

name<String>

Template Name.
Minimum length = 1
Maximum length = 128

id<String>

Id is system generated key for all the VM Templates.

device_family<String>

Device family.
Minimum length = 1
Maximum length = 64

type<String>

Type of device, (Xen | NS).
Minimum length = 1
Maximum length = 64

Read only properties

uuid<String>

UUID of Template.

description<String>

Description of vm_template.
