managed_app

Configuration for Managed Application resource.

Read/write properties

app_networking_data<String>

app networking blob that can be used by placement logic.

is_app_placed<Boolean>

1 if app is placed on some NS, else 0..

container_manager<String>

Container orchestration system.

app_id<String>

Id for the managed application.

app_netfuncs_config<network_function_config[]>

Network function related config for the application.

networking_manager<String>

System used to setup the network.

app_dns<String>

Doman name service for Application.

appname<String>

Name of the Application.

dns_manager<String>

Domain Name Service manager.

id<String>

Dummy Id needed for input get requests.

Read only properties
