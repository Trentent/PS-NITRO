ns_conf_download_policy

Configuration for NetScaler Configuration (ns.conf) Download Policy resource.

Read/write properties

interval_unit<String>

This property is deprecated, polling interval will be provided in minutes only<br>Frequency unit (Minutes).

polling_interval<Integer>

Frequency of polling in minutes.
Minimum value = 30
Maximum value =

ns_ip_address<String[]>

List of NetScaler IP Address.
Minimum length = 1
Maximum length = 64
