sms_server

Configuration for SMS server properties resource.

Read/write properties

optional1_val<String>

Optional1 Val for the sms server.
Maximum length = 128

to_seperater<String>

To list seperater for the sms server.
Maximum length = 128

username_val<String>

Username val for the sms server.
Maximum length = 128

to_key<String>

To key for the sms server.
Maximum length = 128

id<String>

Id is system generated key for all the sms server.

is_ssl<Boolean>

Is SSL support configured..

base_url<String>

Base URL for the sms server, without payload.
Maximum length = 2000

username_key<String>

Username key for the sms server.
Maximum length = 128

optional1_key<String>

Optional1 key for the sms server.
Maximum length = 128

message_key<String>

Message key for the sms server.
Maximum length = 128

message_word_sperater<String>

Message Word Seperater for the sms server.
Maximum length = 128

type<String>

HTTP type supported for the sms server.
Maximum length = 128

password_val<String>

Password Val for the sms server.
Maximum length = 128

password_key<String>

Password key for the sms server.
Maximum length = 128

server_name<String>

SMS server name.
Minimum length = 1
Maximum length = 128

Read only properties

optional2_val1<String>

Optional2 Val for the sms server.

tenant_id<String>

Tenant Id of the Config Jobs.

optional3_key<String>

Optional3 key for the sms server.

optional3_val<String>

Optional3 Val for the sms server.

optional2_key<String>

Optional2 key for the sms server.
