device_generic_resrc

Configuration for Device generic utility resource.

Read/write properties

failure_action<String>

Action taken on failure .

sync<Boolean>

Ping SVM to check its status.

payload<String>

Payload/Body Request .

device_family_type<String>

Device family type.
Minimum length = 1

ipaddress<String[]>

List of SVM IP Address.
Minimum length = 1
Maximum length = 64

resource_type<String>

Type of resource.
Minimum length = 1

Read only properties

act_id<String>

Activity Id.
