cbwanopt_device_profile

Configuration for CBWANOPT Device Profile resource.

Read/write properties

name<String>

Profile Name.
Minimum length = 1
Maximum length = 128

svm_ns_comm<String>

Communication with Instances.
Minimum length = 1
Maximum length = 10

use_global_setting_for_communication_with_ns<Boolean>

True, if the communication with Instance needs to be global and not device specific.

id<String>

Id is system generated key for all the device profiles.

tenant_id<String>

Tenant Id of device profile.
Minimum length = 1
Maximum length = 128

password<String>

Password for this profile.
Minimum length = 1
Maximum length = 127

username<String>

User Name for this profile.
Minimum length = 1
Maximum length = 127

Read only properties

is_default<Boolean>

True, if this profile is system generated and can not be deleted.
