sdx_backup_restore

Configuration for Backup resource.

Read/write properties

ip_address_list<String[]>

List of VM IP Address.
Minimum length = 1
Maximum length = 1024

reset_type<Integer>

Reset Type [0: Reset (Without Network Configuration), 1: Reset (With Network Configuration), 2: Appliance Reset, 3: Appliance Restore, 4: Instance Restore, 5: Backup ].

file_name<String>

File Name.
Minimum length = 1
Maximum length = 256

Read only properties

name<String>

Name of VM device.

act_id<String>

Activity Id.

ip_address<String>

IP Address for this VM device.

type<String>

Type of device, (br | ns).

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.

The following parameters can be used in the nitro request :

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
