ns_lbvserver

Configuration for NS Lbvserver Information resource.

Read/write properties

user_managed<Boolean>

User Managed Idenfication.

hostname<String>

Hostname of the managed device.
Minimum length = 1
Maximum length = 256

ns_ip_address<String>

NS IP Address.
Minimum length = 1
Maximum length = 64

id<String>

Id is system generated key.

vsvr_port<Integer>

Vserver Port.
Minimum value = 1
Maximum value =

name<String>

Vserver Name.
Minimum length = 1
Maximum length = 100

managed<Boolean>

Managed.

vsvr_ip_address<String>

Vserver IP Address.
Minimum length = 1
Maximum length = 64

save_config<Boolean>

Save configuration after service enable/disable.

svc_name<String>

Service Name.
Minimum length = 1
Maximum length = 100

svc_grp_name<String>

Service Group Name.
Minimum length = 1
Maximum length = 100

Read only properties

vsvr_state<String>

Vserver Effective State.

vsvr_state_timestamp<Integer>

Vserver Last State Change Time.

state<String>

Vserver State.

lbmethod<String>

lbmethod.

activeservices<String>

Active services bound to the LB Vserver.

vsvr_type<String>

Vserver Type.

totalservices<String>

Total services bound to the LB Vserver.

vsvr_health<Integer>

Vserver Health.

poll_time<Integer>

Last Polling Time.

display_name<String>

Display Name.

persistencetype<String>

Persistence Type.

comment<String>

Vserver State.

partition_name<String>

Partition Name.

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
