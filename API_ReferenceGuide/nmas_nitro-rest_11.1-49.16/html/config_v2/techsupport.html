techsupport

Configuration for Technical Support resource.

Read/write properties

mode<String>

Technical support Mode, Possible values Appliance | XenServer | Management Service | NetScaler Insight | Appliance_full | Instances.
Minimum length = 1
Maximum length = 128

file_location_path<String>

File Location on Client for upload/download.

file_name<String>

Technical support File Name.
Minimum length = 1
Maximum length = 256

vpx_list_for_techsupport<String[]>

List of VPX for which the techsupport is required (Applicable for only : ManagementServiceIncludingInstances and ApplianceIncludingInstances).
Minimum length = 1
Maximum length = 50

Read only properties

file_size<Double>

File Size.

file_last_modified<String>

Last Modified Time.

act_id<String>

Activity Id.

Default value: EXIT
Possible values = EXIT, CONTINUE
