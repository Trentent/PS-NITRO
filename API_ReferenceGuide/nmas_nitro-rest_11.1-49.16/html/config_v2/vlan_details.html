vlan_details

Configuration for VLAN details resource.

Read/write properties

Read only properties

vm_ip_address<String>

IP Address of the VM where VLAN is configured.

vm_name<String>

Name of the VM where VLAN is configured.

interface_list<String>

List of interface associated with VLAN on the VM.

vlan_id<Integer>

VLAN ID.
