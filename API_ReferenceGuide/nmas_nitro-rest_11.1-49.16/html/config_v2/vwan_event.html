vwan_event

Configuration for VWAN EVENT resource.

Read/write properties

id<String>

Id is system generated key for all the events.

Read only properties

source<String>

Source.

wan_link_state<String>

Internal Event ID used in the source device..

virtual_path<String>

Operation Type.

status_message<String>

status_message.

rpt_sample_time<Double>

Event Time.

virtual_path_state<String>

virtual_path_state.

message<String>

VWAN Event_Message.

host_name<String>

Assign hostname to managed device, if this is not provided, name will be set as host name .

type<String>

type.

category<String>

Category of VWAN Event.

severity<String>

Severity of Vwan Event.

wan_link<String>

WAN Link.
