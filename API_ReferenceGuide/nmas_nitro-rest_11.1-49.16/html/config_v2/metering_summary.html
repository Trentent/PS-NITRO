metering_summary

Configuration for Metering Data Summary resource.

Read/write properties

number_of_bursts<Integer>

Number of bursts by VM Instance in Interval.

ip_address<String>

IP Address for this VM device for which metering data summary is required.
Minimum length = 1
Maximum length = 64

Read only properties

max_burst<Double>

Maximum Burst Throughput of VM Instance for Interval in Mbps.

min_rx<Double>

Minimum In Throughput of VM Instance for Interval in Mbps.

start_time<Double>

Start Time of interval for which summary is generated.

duration<String>

Interval Duration for which summary is generated.

total_txmbits<Double>

Out Total Rx of VM Instance for Interval in Mbits.

total_rxmbits<Double>

In Total Rx of VM Instance for Interval in Mbits.

end_time<Double>

End Time of interval for which summary is generated.

max_rx<Double>

Maximum In Throughput of VM Instance for Interval in Mbps.

max_tx<Double>

Maximum Out Throughput of VM Instance for Interval in Mbps.

min_tx<Double>

Minimum Out Throughput of VM Instance for Interval in Mbps.
