ns_filterpolicy_template

Configuration for NS Filter Policy Configure Task resource.

Read/write properties

priority<Integer>

Policy Priority.

device_groups<String[]>

Device Group Array on which for which template is applied.

name<String>

Name of Policy.

variables<variable_values[]>

Values of Variables used in commands of the configuration template.

state<String>

State of the Policy: enabled/disabled.

devices<String[]>

Device IP Address Array on which template is applied.

actionType<String>

Action Type of the Policy.

actionName<String>

Action Name of the Policy.

expression<String>

Expression of the Policy.

Read only properties
