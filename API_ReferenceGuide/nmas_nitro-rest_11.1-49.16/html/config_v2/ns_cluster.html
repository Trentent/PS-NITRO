ns_cluster

Configuration for ns_cluster resource.

Read/write properties

add_nodegroup<Boolean>

Add Node group in cluster instance if this property is true.

backplane<String>

Backplane Interface.

clusterid<Integer>

Cluster Id.
Minimum value = 1
Maximum value =

nodeid<Integer>

Node Id.
Maximum value =

state<String>

Node State.
Minimum length = 1
Maximum length = 32

password<String>

Password of the Configuration Coordinator node(Cluster IP).
Minimum length = 1
Maximum length = 128

process_local<Boolean>

Process Local.

clviewleader<String>

Cluster CCO NSIP.
Minimum length = 1
Maximum length = 64

inc_mode<Boolean>

Inc Mode.

clnumnodes<Integer>

Cluster ID.
Maximum value =

clnodes<String>

List of Nodes part of Cluster.

preemption<Boolean>

Preemption.

iscco<Boolean>

Is CCO.

clip<String>

Cluster IPAddress.
Minimum length = 1
Maximum length = 64

node_group<String>

Node group.

nodegroup_list<String[]>

Available Node group list on this cluster.

ipaddress<String>

NS ipaddress.
Minimum length = 1
Maximum length = 64

profile_name<String>

Profile Name.
Maximum length = 128

Read only properties

mgmt_cpu_usage<Double>

Management CPU Usage (%) of Cluster Instance.

totpropagationtimeout<Double>

Number of times the update to the client timed-out.

totsteeredpkts<Double>

Total number of packets steered on the cluster backplane.

act_id<String>

Activity Id.

numdfddroppkts<Double>

Total number of packets steered on the cluster backplane.

http_req<Double>

HTTP Requests/second on Cluster Instance.

cpu_usage<Double>

CPU Usage (%) of Cluster Instance.

clbkplanerx<Double>

Traffic received on backplane (in mbits).

clbkplanetx<Double>

Traffic transmitted from backplane (in mbits).

operationalstate<String>

Cluster Operational State.

tx<Double>

Out Throughput of Cluster Instance in Mbps.

clcurstatus<String>

State of the cluster.

rx<Double>

In Throughput of Cluster Instance in Mbps.

memory_usage<Double>

Memory Usage (%) of Cluster Instance.

clbkplanetxrate<Double>

Traffic rate transmitted from backplane (in mbits).

health<String>

Node Health State.

clbkplanerxrate<Double>

Traffic rate received on backplane (in mbits).
