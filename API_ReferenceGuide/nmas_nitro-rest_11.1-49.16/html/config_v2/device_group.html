device_group

Configuration for Device Group resource.

Read/write properties

static_device_list<String>

Devices in the group.

device_family<String>

Device Family.
Minimum length = 1
Maximum length = 64

name<String>

Device Group Name.
Maximum length = 255

criteria_type<String>

Device Group Criteria.
Maximum length = 255

criteria_condn<String>

Tenant.
Maximum length = 255

id<String>

Id is system generated key for agent registered with NMX Cloud..

criteria_value<String>

Criteria Value.
Maximum length = 255

static_device_list_arr<String[]>

Static Device List.

Read only properties

tenant_id<String>

Tenant Id.
