ns_crvserver

Configuration for NS CRvserver Information resource.

Read/write properties

vsvr_port<Integer>

Vserver Port.
Minimum value = 1
Maximum value =

user_managed<Boolean>

User Managed Idenfication.

name<String>

Vserver Name.
Minimum length = 1
Maximum length = 100

ns_ip_address<String>

NS IP Address.
Minimum length = 1
Maximum length = 64

managed<Boolean>

Managed.

vsvr_ip_address<String>

Vserver IP Address.
Minimum length = 1
Maximum length = 64

id<String>

Id is system generated key.

save_config<Boolean>

Save configuration after service enable/disable.

Read only properties

hostname<String>

Hostname of the managed device.

poll_time<Integer>

Last Polling Time.

display_name<String>

Display Name.

state<String>

Vserver State.

cachetype<String>

Cache Type.

hits<String>

Vserver Health.

partition_name<String>

Partition Name.

vsvr_type<String>

Vserver Type.

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.

The following parameters can be used in the nitro request :

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
