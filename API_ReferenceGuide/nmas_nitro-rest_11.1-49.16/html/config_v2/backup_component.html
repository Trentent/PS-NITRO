backup_component

Read/write properties

component_message<String>

Information about the componenet.

component_minimum_version<String>

Minimum version supported by the NetScaler.

component_name<String>

Backup component Name.

component<String>

Backup component.

backup_file_name<String>

Backup file name.
Maximum length = 64

component_versions<String>

Backup component version.

component_ip<String>

Backup component IPaddress.

component_error_severity<Integer>

Severity of error for the component.

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
