license_server_pool

Configuration for License pool capacity of license server resource.

Read/write properties

instance_available<Double>

Available instance pool.

Read only properties

std_bw_available<Double>

Available Standard Bandwidth pool in Gbps.

plt_bw_available<Double>

Available Platinum Bandwidth pool in Gbps.

std_bw_total<Double>

Total Standard Bandwidth pool in Gbps.

ent_bw_available<Double>

Available Enterprise Bandwidth pool in Gbps.

ent_bw_total<Double>

Total Enterprise Bandwidth pool in Gbps.

instance_total<Double>

Total instance pool.

plt_bw_total<Double>

Total Platinum Bandwidth pool in Gbps.
