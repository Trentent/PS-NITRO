metering_data

Configuration for Metering Data for VM Devices resource.

Read/write properties

id<String>

Id is system generated key for all the metering data points.

ip_address<String>

IP Address for this VM device for which data is recorded.
Minimum length = 1
Maximum length = 64

Read only properties

interval_txmbits<Double>

Out Tx of VM Instance for Interval in Mbits.

burst_thpt<Double>

Burst Throughput of VM Instance in Mbps.

tx<Double>

Out Throughput of VM Instance in Mbps.

rx<Double>

In Throughput of VM Instance in Mbps.

start_time<Double>

Start Time of Interval for which data was recorded.

min_thpt<Double>

Minimum Throughput allocated to Instance in Mbps.

interval_rxmbits<Double>

In Rx of VM Instance for Interval in Mbits.

poll_time<Double>

Poll Time at which data was recorded.

max_thpt_limit<Double>

Maximum Throughput Limit allowed for VM Instance in Mbps.

total_txmbits<Double>

Out Total Rx of VM Instance in Mbits.

thpt_limit<Double>

Throughput Limit configured for VM Instance in Mbps.

total_rxmbits<Double>

In Total Rx of VM Instance in Mbits.
