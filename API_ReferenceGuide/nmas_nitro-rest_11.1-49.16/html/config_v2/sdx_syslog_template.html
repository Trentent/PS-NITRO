sdx_syslog_template

Configuration for NS Syslog Configure Task resource.

Read/write properties

scheduleTimesEpoch<String>

Schedule time epoch (string representation of 11 digit numbers)..

device_groups<String[]>

Device Group Array on which for which template is applied.

scheduleTimes<String>

Comma Seperated times of the day(HH:MM) on which Configuration Template is scheduled.

name<String>

Name of Syslog Server.
Minimum length = 1
Maximum length = 128

variables<variable_values[]>

Values of Variables used in commands of the configuration template.

on_error<String>

Behaviour on encountering error while applying configuration template on a device: CONTINUE|EXIT.
Minimum length = 1
Maximum length = 16

port<Integer>

Syslog server port.
Maximum value =

execute_sequentially<Boolean>

True if configuration template is to be applied to devices sequentially.

tz_offset<Integer>

Time zone offset..

scheduleOptions<String>

Comma Seperated Options for scheduling Configuration Template.

devices<String[]>

Device IP Address Array on which template is applied.

scheduleType<String>

Schedule Type of Configuration Template that is scheduled.

ip_address<String>

Syslog server IP address.
Minimum length = 1
Maximum length = 64

Read only properties
