mps_ssl_certkey

Configuration for Install SSL certificate on Management Service resource.

Read/write properties

password<String>

The pass-phrase that was used to encrypt the private-key..
Maximum length = 32

ssl_key<String>

Key.
Minimum length = 1
Maximum length = 128

ssl_certificate<String>

Certificate.
Minimum length = 1
Maximum length = 128

Read only properties

serial_number<String>

Serial Number.

signature_algorithm<String>

Signature Algorithm.

valid_from<String>

Valid From.

status<String>

Tells whether the certificate is still valid or not.

issuer<String>

Issuer.

public_key_size<Integer>

Public Key Size.

valid_to<String>

Valid To.

version<String>

Version.

subject<String>

Subject.

public_key_algorithm<String>

Public Key Algorithm.

days_to_expiry<Integer>

Days before SSL certificate expires.
