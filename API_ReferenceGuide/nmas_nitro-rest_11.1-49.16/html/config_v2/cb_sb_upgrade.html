cb_sb_upgrade

Configuration for CBSDX Single Bundle Upgrade resource.

Read/write properties

file_name<String>

File Name.
Minimum length = 1
Maximum length = 256

ns_ip_arr_to_save<String[]>

List of NetScaler IP address for which config needs to be saved.

Read only properties

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.

The following parameters can be used in the nitro request :

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE

upgrade

upgrade

URL:http://<MGMT-IP>/nitro/v2/config/cb_sb_upgrade?action=upgrade&onerror=<String_value>

HTTP Method:POST

Request Payload:JSON

{cb_sb_upgrade: {
      "file_name":<String_value>,
      "ns_ip_arr_to_save":<String_value[]>}}

Response Payload:JSON

{ "errorcode": 0, "message": "Done", "severity": <String_value>, "cb_sb_upgrade":[{
      "file_name":<String_value>,
      "ns_ip_arr_to_save":<String_value>}]}
