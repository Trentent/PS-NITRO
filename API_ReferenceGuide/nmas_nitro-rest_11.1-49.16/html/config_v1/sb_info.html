sb_info

Configuration for Single Bundle File Info resource.

Read/write properties

file_name<String>

File Name.
Minimum length = 1
Maximum length = 256

Read only properties

component_info<String[]>

Component Info.

error_message<String>

Error Message in case the Single Bundle cannot be applied.

total_approx_time<Integer>

Total approximate time that the upgrade will take.

reboot_message<String>

Reboot Message.

selected_sb_version<String>

Selected Single Bundle Version.

current_sb_version<String>

Current Single Bundle Version.

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.

The following parameters can be used in the nitro request :

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
