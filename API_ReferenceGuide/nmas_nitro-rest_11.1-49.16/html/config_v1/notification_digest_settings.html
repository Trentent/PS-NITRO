notification_digest_settings

Configuration for Notification Digest Settings resource.

Read/write properties

WeeklyFeaturesList<String>

List of events for which notification is to be generated weekly..

scheduleTimesEpoch<String>

Schedule time epoch (string representation of 11 digit numbers)..

tz_offset<Integer>

Time zone offset..

scheduleOptions<String>

Comma Seperated Options for scheduling Configuration Template.

ImmediateFeaturesList<String>

List of events for which notification is to be generated immediately..

MonthlyFeaturesList<String>

List of events for which notification is to be generated monthly..

nextScheduleTimeEpoch<String>

Schedule time epoch at which job is scheduled to be run next..

id<String>

Id is system generated for each setting.

DailyFeaturesList<String>

List of events for which notification is to be generated daily..

notification_period<String>

notification_period.

mail_profile<String>

Mail Profile.

Read only properties

tenant_id<String>

Tenant Id of the Notification Jobs.
