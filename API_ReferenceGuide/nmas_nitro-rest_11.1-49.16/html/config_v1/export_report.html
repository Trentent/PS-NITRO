export_report

Configuration for Export report resource.

Read/write properties

page_name<String>

Page name..
Minimum length = 1
Maximum length = 255

recurrence_option<String>

Recurrence option (Weekly -> mon - sun, Monthly -> date in DD format)..
Maximum length = 128

report_context<String>

Report context..

export_format<String>

Report Export format (PDF/JPEG/PNG).
Minimum length = 3
Maximum length = 4

description<String>

Report Description..
Maximum length = 512

report_title<String>

Report Title.

mail_profile_name<String>

Mail profile name to send notification mail..
Maximum length = 255

recurrence<String>

Recurrence (Daily, Weekly, Monthly)..
Maximum length = 16

tz_offset<Integer>

Time zone offset..

file_name<String>

Report File name..

csv_export_arr<csv_export[]>

CSV Export Array.

export_time<String>

Report Export time epoch (string representation of 11 digit numbers)..
Minimum length = 10
Maximum length = 16

export_now<Boolean>

Export now.

Read only properties

The following parameters can be used in the nitro request :

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
