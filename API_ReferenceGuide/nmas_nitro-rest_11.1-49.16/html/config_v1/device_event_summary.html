device_event_summary

Configuration for devices events summary resource.

Read/write properties

id<String>

Id is system generated key for all the system users.

device_ip_address<String>

ip_address of the device..

type<String>

Device Family.
Minimum length = 1
Maximum length = 64

hostname<String>

Assign hostname to managed device, if this is not provided, name will be set as host name .
Minimum length = 1
Maximum length = 256

Read only properties

total_critical_events<Integer>

Total Critical events city wise..

total_major_events<Integer>

Total major events..

total_events<Integer>

Total events.

instance_state<String>

State of device, UP only if device accessible.
