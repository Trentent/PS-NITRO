triton_config

Configuration for Configuration details for Triton module resource.

Read/write properties

marathon_params<marathon_param>

Marathon parameters.

netmode<String>

URL for the Marathon connection.
Maximum length = 200

dns_params<dns_param>

DNS Parameters.

loopback_disabled<Boolean>

This indicates default monitor is enabled on this app or not.

framework_healthcheck<Boolean>

This indicates default monitor is enabled on this app or not.

id<String>

Id.

networking_params<networking_param>

Networking parameters.

Read only properties

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.

The following parameters can be used in the nitro request :

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
