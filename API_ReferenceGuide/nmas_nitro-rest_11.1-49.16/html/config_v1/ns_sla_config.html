ns_sla_config

Configuration for SLA Virtual server config on NetScaler resource.

Read/write properties

appflowlog<String>

Appflow log.

acceptancethreshold<String>

Tells whether bind type virtual server type.
Minimum length = 1
Maximum length = 10

is_active_policy<Integer>

Policy is Active or Not.

rule<String>

Responder policy rule.

appflowaction<String>

appflowaction.

is_response_time_set<Integer>

is_response_time_set.

policy_action<String>

policy_action NOOP.

ns_ip_address<String>

NS IP Address.
Minimum length = 1
Maximum length = 64

hits<String>

Hits of Responder Policy.

maxtransactionthreshold<String>

rt_max_threshold .

DOI<String>

DOI name.
Maximum length = 255

vserver_ip_address<String>

NetScaler IP Address.
Minimum length = 1
Maximum length = 64

bind_priority<String>

bind_priority 1 to 2147483647.

bind_type<String>

Tells whether bind type virtual server type.
Minimum length = 2
Maximum length = 64

mintransactionthreshold<String>

rt_min_threshold .

time_window_interval<String>

time_window_interval 1 min to 1440 mins .

Read only properties
