mail_profile

Configuration for Mail profile resource.

Read/write properties

to_list<String>

List of to whom send the mail. .
Minimum length = 6

sender_mail_address<String>

Email Address from where mail is send.

bcc_list<String>

List to whom BCC the mail..

id<String>

Id is system generated key for all the mail profile..

cc_list<String>

List to whom CC the mail..

server_name<String>

SMTP server name.
Minimum length = 1
Maximum length = 128

profile_name<String>

Profile name for the mail setting..
Minimum length = 1
Maximum length = 128

Read only properties

tenant_id<String>

Tenant Id of the Config Jobs.

The following parameters can be used in the nitro request :

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
