export_report_job

Configuration for Scheduled export report job resource.

Read/write properties

recurrence_option<String>

Recurrence option (Weekly -> mon - sun, Monthly -> date in DD format)..

status<String>

Schedule Status.
Minimum length = 1
Maximum length = 128

export_format<String>

Report Export format (PDF/JPEG/PNG).
Minimum length = 3
Maximum length = 4

description<String>

Report Description..
Maximum length = 512

mail_profile_name<String>

Mail profile name to send notification mail..
Minimum length = 1
Maximum length = 255

recurrence<String>

Recurrence (Daily, Weekly, Monthly)..
Maximum length = 16

tz_offset<Integer>

Time zone offset..

next_scheduletime<Integer>

Next Schedule Time.

export_time<String>

Report Export time GMT epoch (string representation of 10 digit numbers)..
Minimum length = 5
Maximum length = 16

job_name<String>

job name is ExportJob..
Maximum length = 16

trigger_start<String>

Time at which the trigger should start.Format:YY:MM:DD:HH:MM.

report_name<String>

report name.
Maximum length = 255

id<String>

Id is job_schedule id for all the export_report_job details.

Read only properties
