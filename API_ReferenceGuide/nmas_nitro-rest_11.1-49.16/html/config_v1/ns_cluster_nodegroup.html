ns_cluster_nodegroup

Configuration for ns_cluster_nodegroup resource.

Read/write properties

password<String>

Password of the Configuration Coordinator node(Cluster IP).
Minimum length = 1
Maximum length = 128

clip<String>

Cluster IPAddress where this nodegroup will be added.
Minimum length = 1
Maximum length = 64

name<String>

Cluster Node Group Name.
