ns_snmp_config

Configuration for SNMP configuration resource.

Read/write properties

save_config<Boolean>

true, if save config is required.

trapdestination<String>

State of configuration (enable/disable).

snmpauthprotocol<String>

SNMP v3 auth protocol.

snmpsecurityname<String>

SNMP v3 security name.
Maximum length = 31

snmpprivprotocol<String>

SNMP v3 priv protocol.

ns_ip_address_arr<String[]>

List of NS IP Address.
Maximum length = 64

state<String>

State of configuration (enable/disable).

ns_ip_address<String>

NetScaler IP Address.
Minimum length = 1
Maximum length = 64

snmpsecuritylevel<String>

SNMP v3 security level.

snmpcommunity<String>

SNMP community.
Maximum length = 31

snmpprivpassword<String>

SNMP v3 priv password.
Minimum length = 8
Maximum length = 31

snmpversion<String>

SNMP version.

snmpauthpassword<String>

SNMP v3 auth password.
Minimum length = 8
Maximum length = 31

Read only properties
