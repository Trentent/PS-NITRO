backup_policy

Configuration for Backup Policy resource.

Read/write properties

backup_to_retain<Integer>

Number of previous backups to retain.
Minimum value = 1
Maximum value =

enable_external_transfer<Boolean>

Enable transfer of backup file to external server.

policy_name<String>

Policy Name.
Minimum length = 1
Maximum length = 128

external_storage_info<backup_external_storage>

Information of the External storage for backup file.

backup_password<String>

Password for backup file encryption.

encrypt_backup_file<Boolean>

Encrypts backup files.

id<String>

Id is system generated key.

Read only properties

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.

The following parameters can be used in the nitro request :

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
