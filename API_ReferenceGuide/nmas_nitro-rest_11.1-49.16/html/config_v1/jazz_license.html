jazz_license

Configuration for Jazz License Information resource.

Read/write properties

date_purchased<String>

Purchase Date for license.

session_id<String>

Session id for license.

features<String>

Features for license.

relevance<Integer>

Relevance of license.

name<String>

Name of license.

serial_no<String>

Serial Number for license.

count_total<Integer>

Count Total of license.

count_available<Integer>

Count Available of license.

id<String>

ID for license.

date_exp<String>

Expiry Date for license.

bind_type<String>

Bind Type for license.

use_proxy<Boolean>

Use license proxy configured to connect to citrix backoffice.

Read only properties

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
