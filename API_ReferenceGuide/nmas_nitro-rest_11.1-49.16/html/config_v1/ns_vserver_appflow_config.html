ns_vserver_appflow_config

Configuration for Virtual server appflow config on NetScaler resource.

Read/write properties

export_option<String[]>

Export Options: TCP, ICA.

appflow_policy_rule<String>

Appflow policy rule.

appflowlog<String>

Appflow log.

es4nslog<String>

ESNS enable.

name<String>

Virtual server name.
Maximum length = 255

export_list<String[]>

Export List: WebInsight, SecurityInsight.

ns_ip_address<String>

NetScaler IP Address.
Minimum length = 1
Maximum length = 64

agent_list<String[]>

Agent List, on which traffic will flow.

ip_address<String>

Virtual server IP Address.
Minimum length = 1
Maximum length = 64

type<String>

Tells whether virtual server type.
Minimum length = 2
Maximum length = 64

icalog<String>

ICA log.

servicetype<String>

servicetype.

Read only properties

state<String>

Tells whether virtual server up/down.

Default value: EXIT
Possible values = EXIT, CONTINUE
