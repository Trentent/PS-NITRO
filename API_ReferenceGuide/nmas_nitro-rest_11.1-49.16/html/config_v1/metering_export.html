metering_export

Configuration for Metering Data Export resource.

Read/write properties

file_name<String>

File Name.
Minimum length = 1
Maximum length = 512

ip_address<String>

IP Address for this VM device for which metering data export is required.
Minimum length = 1
Maximum length = 64

Read only properties

end_time<Double>

End Time of interval for which export is generated.

file_size<Integer>

file_size.

start_time<Double>

Start Time of interval for which export is generated.

file_last_modified<String>

Last Modified.

duration<String>

Interval Duration for which export is generated.
