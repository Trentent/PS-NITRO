ns_l7_latency_config

Configuration for L7 Latency configuration on NetScaler resource.

Read/write properties

vserverType<String>

Virtual server type(lb/sslvpn/CR).
Minimum length = 2
Maximum length = 64

l7LatencyThresholdNotifyIntetval<Integer>

L7 Latency Threshold Notify Intetval.

l7LatencyNotifyCount<Integer>

L7 Latency Notify Count.

l7LatencyThresholdWaitInterval<Integer>

L7 Latency Threshold Wait Interval.

ns_ip_address<String>

NetScaler IP Address.
Minimum length = 1
Maximum length = 64

l7LatencyThresholdFactor<Integer>

L7 Latency Threshold Factor Value.

vserverName<String>

Virtual server name.
Maximum length = 255

l7LatencyMonitoring<String>

ENABLE/DISABLE.
Maximum length = 512

Read only properties

latencyName<String>

Name of latency profile.
