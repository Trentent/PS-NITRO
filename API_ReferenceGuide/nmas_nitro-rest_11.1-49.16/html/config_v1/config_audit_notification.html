config_audit_notification

Configuration for Config Audit Notification configuration resource.

Read/write properties

name<String>

name.

rule<String>

Rule.

sms_profile<String>

SMS Profile.

feature<String>

feature.

id<String>

Id is system generated key for all the Thresholds configuration .
Maximum length = 256

resource_type<String>

Resource Type.

mail_profile<String>

Mail Profile.

is_enabled<String>

true or false.

duration<String>

duration of metric to be checked against threshold.

tenant_name<String>

Tenant Name.

group_name<String>

Group Name.

reference_key<String>

Reference Key.

Read only properties
