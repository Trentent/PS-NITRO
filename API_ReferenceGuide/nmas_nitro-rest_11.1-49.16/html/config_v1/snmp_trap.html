snmp_trap

Configuration for SNMP Trap Destinations resource.

Read/write properties

community<String>

Community Name.
Maximum length = 32

version<String>

SNMP version.
Maximum length = 2

dest_server<String>

Trap Destination Server Address.

dest_port<Integer>

Destination Port.
Minimum value = 1
Maximum value =

user_name<String[]>

Name of SNMP Trap User.
Minimum length = 1
Maximum length = 32

Read only properties
