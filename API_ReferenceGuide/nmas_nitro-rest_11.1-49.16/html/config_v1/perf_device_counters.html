perf_device_counters

Configuration for Device Counter Table resource.

Read/write properties

device_family<String>

Device Family of which the counter belongs to.

counter_group<String>

Counter Group of which the counter belongs to.
Minimum length = 1
Maximum length = 100

is_tabular<Boolean>

Is Tabular.

min_ver_supported<String>

Minimum Version supported by the counter Group.
Minimum length = 1
Maximum length = 100

counter_name<String>

Counter Name.
Minimum length = 1

Read only properties
