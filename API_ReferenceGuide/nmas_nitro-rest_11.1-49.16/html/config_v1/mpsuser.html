mpsuser

Configuration for System User resource.

Read/write properties

session_timeout<Integer>

Session timeout for the user.

permission<String>

This property is not supported, permission will be taken from groups only<br>Actions that this user is authorized to perform.
Maximum length = 128

name<String>

User Name.
Minimum length = 1
Maximum length = 128

session_timeout_unit<String>

Session timeout unit for the user.

external_authentication<Boolean>

Enable external authentication.

enable_session_timeout<Boolean>

Enables session timeout for user.

tenant_id<String>

Tenant Id of the system users.
Minimum length = 1
Maximum length = 128

password<String>

Password.
Minimum length = 1
Maximum length = 128

id<String>

Id is system generated key for all the system users.

groups<String[]>

Groups to which user belongs.

Read only properties
