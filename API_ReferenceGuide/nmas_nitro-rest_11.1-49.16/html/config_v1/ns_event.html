ns_event

Configuration for NetScalerEvent resource.

Read/write properties

cmd_exec_status<String>

Command Execution Status.
Maximum length = 255

device_entity_type<String>

Device Entity Type.
Maximum length = 255

device_type<String>

Device Type.
Minimum length = 1
Maximum length = 64

id<String>

Id is system generated key for all the events.

device_family<String>

Device Family.

rpt_sample_time<Double>

Report Sample time..

user_name<String>

Username.
Maximum length = 255

device_entity_name<String>

Device Entity Name.

config_cmd<String>

Config Command.

cmd_auth_status<String>

Command Authorization Status.
Maximum length = 255

hostname<String>

Assign hostname to managed device, if this is not provided, name will be set as host name .
Minimum length = 1
Maximum length = 256

Read only properties

source<String>

Source.

trap_id<Integer>

Last Octet of SNMP Trap OID.

entity<String>

Entity of Event.

timestamp<Integer>

Event Time.

operation_type<String>

Operation Type.

category<String>

Category of Event.

severity<String>

Severity of Event.

failureobj<String>

Failure Object.

source_event_id<Integer>

Internal Event ID used in the source device..

history<String>

History of the Event with same entity.

message<String>

Event Message.
