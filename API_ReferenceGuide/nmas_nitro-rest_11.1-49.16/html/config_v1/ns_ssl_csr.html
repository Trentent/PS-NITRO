ns_ssl_csr

Configuration for CSR File resource.

Read/write properties

tenant_name<String>

Tenant Name.

commonname<String>

Common Name.
Minimum length = 1

statename<String>

State Name.
Minimum length = 2
Maximum length = 128

organizationname<String>

Organization Name.
Minimum length = 1

keyform<String>

Key Form.

file_location_path<String>

File Location on Client for download.
Minimum length = 1

city<String>

City.

file_name<String>

Name for and, optionally, path to the certificate signing request (CSR).
Minimum length = 1
Maximum length = 256

countryname<String>

Country Name.
Minimum length = 2
Maximum length = 128

csr<String>

Generated CSR.

keyfile<String>

Key File.
Minimum length = 1
Maximum length = 128

passphrase<String>

Pass Phrase.

Read only properties

file_size<Integer>

file_size.

file_last_modified<String>

Last Modified.
