device_license_info

Configuration for Device License Info resource.

Read/write properties

last_updated_time<Double>

Last Updated Time.

Read only properties

configured_on_vm<Integer>

Configured license capacity of VM.

current_throughput<String>

Current Througput of device.

license_pool<String>

License Pool.

allocated_on_vm<Integer>

Running license capacity of VM.

name<String>

Name of managed device.

days_to_expiry<Integer>

No of days after which grace license expires.

grace<Boolean>

True if this device is managed by NMX.

managed_device_id<String>

Device id managaed device.

license_state<String>

State of device, UP only if device accessible.

allocated<Integer>

Allocate license on License server.

ip_address<String>

IP Address for this managed device.

type<String>

Type of device, (Xen | NS).
