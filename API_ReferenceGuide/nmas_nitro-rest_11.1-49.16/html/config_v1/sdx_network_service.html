sdx_network_service

Configuration for SDX Network Service resource.

Read/write properties

gateway<String>

Default Gateway of managed device.
Minimum length = 1
Maximum length = 64

network_interfaces<network_interface[]>

Network Interfaces.

vm_memory_total<Double>

Total Memory of VM Instance in MB.
Minimum value = 1024

id<String>

Id is system generated key for all the SDX Network Services.

ip_address<String>

IP Address for this managed device.
Minimum length = 1
Maximum length = 64

netmask<String>

Netmask of managed device.
Minimum length = 1
Maximum length = 64

number_of_cores<Integer>

Number of cores that are assigned to VM Instance.

name<String>

Network Service Name.
Minimum length = 1
Maximum length = 128

description<String>

description.
Minimum length = 1
Maximum length = 512

formation_instance_id<String>

Formation Instance Id.
Minimum length = 1
Maximum length = 128

number_of_ssl_cores<Integer>

Assign number of ssl virtual functions to VM Instance.

type<String>

Service Type.
Minimum length = 1
Maximum length = 64

Read only properties

uuid<String>

UUID of VM Instance.

disk_allocation<String>

Disk allocation for VM Instance.

instance_state<String>

State of device, UP only if device accessible.

disk_space<Double>

Show Disk Space (GB) available to VM Instance.
