ns_ssl_certkey

Configuration for SSL certificate on NetScaler resource.

Read/write properties

no_domain_check<Boolean>

Specify this option to override the check for matching domain names during certificate update operation.

ns_ip_address<String>

List of NetScaler IP Address.
Minimum length = 1
Maximum length = 64

certkeypair_name<String>

Cert Key Pair Name.
Minimum length = 1
Maximum length = 128

ssl_key<String>

Key.
Minimum length = 1
Maximum length = 128

id<String>

Id is system generated key for all ssl cert-keys entries. For download operation "id" must be provided in the format <ns_ip_address>_<certkeypair_name>.tgz.

key_data<String>

Key Data.
Maximum length = 16384

ssl_certificate<String>

Certificate.
Minimum length = 1
Maximum length = 128

certificate_data<String>

Certificate Data.
Maximum length = 16384

file_location_path<String>

File Location on Client for download.
Minimum length = 1

cert_format<String>

Certificate Format.
Maximum length = 64

certchainbinding<String[]>

Certificate Chain binding..

save_config<Boolean>

true, if save config is required.

password<String>

The pass-phrase that was used to encrypt the private-key..
Maximum length = 128

ns_ip_address_arr<String[]>

List of NetScaler IP Address.
Minimum length = 1
Maximum length = 64

Read only properties

serial_number<String>

Serial Number.

signature_algorithm<String>

Signature Algorithm.

valid_from<String>

Valid From.

status<String>

Tells whether the certificate is still valid or not.

hostname<String>

Host Name of the device.

issuer<String>

Issuer.

public_key_size<Integer>

Public Key Size.

device_name<String>

Name of the device.

valid_to<String>

Valid To.

subject<String>

Subject.

version<Integer>

Version.

public_key_algorithm<String>

Public Key Algorithm.

days_to_expiry<Integer>

Days before SSL certificate expires.

poll_time<Integer>

Last Polling Time.

display_name<String>

Display Name of the device.

no_of_bound_entities<Integer>

no_of_bound_entities.

partition_name<String>

Name of Admin Partition. Blank means Default Partition.

csr<String>

Certificate Signing Request.

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.

The following parameters can be used in the nitro request :

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
